### **üî¥ CRITICAL VIOLATIONS (Instant Rejection Risk)**

#### **1.1 Author Identity Leaks**

**‚ùå PROHIBITED:**
```latex
% DO NOT WRITE:
"We conducted a study at Stanford University..."
"Our team at Google Research developed..."
"Building on our previous work [Smith et al., 2023]..."
"We collected data using our internal annotation platform..."
"Participants were recruited from our institution..."
```

**‚úÖ CORRECT:**
```latex
% WRITE INSTEAD:
"We conducted a study at a large research university..."
"Researchers at a technology company developed..."
"Building on prior work [23]..." (cite normally, don't say "our")
"We collected data using a custom annotation platform..."
"Participants were recruited from a university community..."
```

#### **1.2 Self-Citation Anonymization**

**Strategy Matrix:**

| Your Citation | Anonymization Strategy | Example |
|---------------|------------------------|---------|
| **Recent unpublished work** | Omit or cite as "Anonymous" | [Anonymous, 2024] or "Concurrent work investigates..." |
| **Published work (>1 year)** | Cite normally, third person | "Prior work [15] showed..." (not "Our prior work") |
| **Published work (recent)** | Cite normally but avoid clustering | Distribute among other citations |
| **ArXiv preprints** | Avoid if possible | Wait to post until after review |
| **Workshop papers** | Cite normally | Treat as any other citation |

**Example Rewrites:**
```latex
% ORIGINAL (reveals identity):
In our previous study [Smith et al., 2023], we found that users 
prefer interface A. We now extend this work to interface B.

% ANONYMIZED VERSION 1 (if citation necessary):
Prior research [23] found that users prefer interface A. We extend 
this line of work to interface B.

% ANONYMIZED VERSION 2 (if very recent/identifying):
Recent work has shown user preferences for interface A [citation 
omitted for anonymity]. We extend this investigation to interface B.

% ANONYMIZED VERSION 3 (if unpublished):
Concurrent work investigates interface A [Anonymous, 2024]. We 
focus on interface B and provide comparative analysis.
```

#### **1.3 Institutional References**

**Common Leaks:**

| ‚ùå Identifying | ‚úÖ Anonymous | Notes |
|----------------|--------------|-------|
| "Stanford IRB Protocol #12345" | "University IRB Protocol [redacted]" | Remove protocol numbers |
| "MIT CSAIL participants" | "Computer science students" | Remove department names |
| "Recruited via [University] mailing list" | "Recruited via university mailing list" | Lowercase, generic |
| "Amazon Mechanical Turk" | "A crowdsourcing platform" | First mention only, then "MTurk" OK |
| "Prolific Academic" | "An online participant recruitment platform" | Generic description |
| "Google Forms" | "An online survey platform" | Generic unless critical to method |
| "Zoom interviews" | "Video conferencing software" | Generic |

**IRB Anonymization Template:**
```latex
% ORIGINAL:
This study was approved by the Stanford University Institutional 
Review Board (Protocol #45123). Participants were recruited through 
the Stanford Psychology Department subject pool.

% ANONYMIZED:
This study was approved by the Institutional Review Board at [a large 
research university] (Protocol #[redacted]). Participants were 
recruited through a university psychology subject pool.

% ALTERNATIVE (more generic):
This study received ethical approval from an institutional review 
board. Participants were recruited from a university participant pool 
with informed consent.
```

**Screenshot Preparation Script:**
```python
# Use this to batch-process screenshots
from PIL import Image, ImageDraw, ImageFont

def anonymize_screenshot(image_path, regions_to_blur):
    """
    Blur specific regions of screenshot
    
    regions_to_blur: list of (x1, y1, x2, y2) coordinates
    """
    img = Image.open(image_path)
    draw = ImageDraw.Draw(img)
    
    for region in regions_to_blur:
        # Draw black rectangle over identifying info
        draw.rectangle(region, fill='black')
    
    # Add anonymization notice
    font = ImageFont.load_default()
    draw.text((10, 10), "[Institution name redacted for review]", 
              fill='red', font=font)
    
    img.save(image_path.replace('.png', '_anon.png'))

# Example usage:
anonymize_screenshot(
    'interface_screenshot.png',
    regions_to_blur=[
        (0, 0, 200, 50),      # Title bar
        (800, 0, 1000, 30),   # Email address
        (50, 500, 300, 550)   # Logo
    ]
)
```

#### **3.3 Filename & Directory Anonymization**

**Check for Identifying Names:**
```bash
# Scan for identifying strings in code/data
grep -r "Stanford" ./code/
grep -r "smith@university.edu" ./code/
grep -r "internal-project-name" ./code/

# Common places to check:
# - File headers/comments
# - Configuration files
# - Log outputs
# - Data file paths
# - Import statements
# - Documentation
```

**Anonymization Script:**
```python
#!/usr/bin/env python3
import os
import re

IDENTIFYING_PATTERNS = [
    r'Stanford',
    r'MIT',
    r'smith@',
    r'john\.doe',
    r'internal-tool-name',
    r'project-codename'
]

def anonymize_file(filepath):
    """Remove identifying information from file"""
    with open(filepath, 'r', encoding='utf-8', errors='ignore') as f:
        content = f.read()
    
    original_content = content
    
    for pattern in IDENTIFYING_PATTERNS:
        content = re.sub(pattern, '[REDACTED]', content, flags=re.IGNORECASE)
    
    if content != original_content:
        print(f"Anonymized: {filepath}")
        with open(filepath, 'w', encoding='utf-8') as f:
            f.write(content)

# Scan all files
for root, dirs, files in os.walk('./code'):
    for file in files:
        if file.endswith(('.py', '.txt', '.md', '.yaml', '.json')):
            anonymize_file(os.path.join(root, file))
```
